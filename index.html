Quick Start

# Clone
$ git clone https://github.com/kcjsa/ForwardOS_kernel_base
$ cd ForwardOS_kernel_base

# Build kernel
$ cd kernel
$ make run 


$ cd ..

$ cd bootloader 

$ make run

Requirements

- Rust nightly
- qemu-system-x86_64
- cargo-binutils
